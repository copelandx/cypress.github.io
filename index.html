Bot Commands

  • Moderation
  • Public
  • Information
  • Command | Optional
    Summary
  • /clear amount 20 target@James
    Clear a spefific amount of messages from a target or channel
  • /createverify Channel #welcome
    Set your verification channel | MUST have welcome role set
  • /mute target@James time30m reasonReason here
    Mute a member from this guild
  • /say messageBeep Boop color#FFF / white
    pretend to the be the bot :3
  • /setup-suggest channel#suggest
    Point to a channel to handle all suggestions !
  • /setup-tickets categoryTickets
    Point to a category to handle all incoming tickets
  • /setup-welcome channel#welcome
    Set up your welcome channel for the bot | Leave blank to turn off welcome messages
  • /setup-welcome-msg messageWelcome [member]
    Set up your welcome message for the bot
  • /setup-welcome-role welcome-role@NewComer
    Set up a role for new comers !
  • /ticket eventsStart
    Enables Ticketing events | Run /setup-tickets before this command
  • /ticket eventsTurn off
    Disables users from creating tickets
  • /ticket eventsAdmin
    Resolve or Close a ticket | Must be in a ticket channel to run this
  • /unban userid12345678
    Unban a user from the discord server
